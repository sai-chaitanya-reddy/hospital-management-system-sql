# Hospital Management System Database (SQL)

**Production-Ready SQL Schema for Hospital Workflow Management**

## Overview
This project contains a complete Hospital Management System relational database designed using SQL.
It models real-world hospital workflows including patient records, doctors, appointments, billing, and departments.

## Database Features
- Normalized relational schema
- Primary and foreign key constraints
- Realistic healthcare workflow modeling
- Scalable and maintainable design

## Tables Included
- Patients
- Doctors
- Appointments
- Departments
- Billing
- Staff
- Medical Records

## Use Cases
- SQL interview practice
- Data analyst portfolio project
- Backend system design learning
- Healthcare data modeling
- Academic DBMS projects

## Technologies Used
SQL, MySQL, PostgreSQL, Relational Database, DBMS

## How to Use
1. Clone the repository
2. Import the SQL file into MySQL or PostgreSQL
3. Execute the schema
4. Run queries for analysis or backend development
